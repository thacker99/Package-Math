 ***** Projet : Package Math Personnel *****

Ce projet vise à créer et à regrouper un ensemble de fonctions mathématiques pour une utilisation personnelle.

Le projet est codé en Java et est constitué de deux fichiers:

- Fonctions.java
- TestFonctions.java

Le fichier Fonctions.java regroupe un ensemble de fonctions mathématiques tandisque le fichier TestFonctions.java utilise les
fonctions mathématiques contenues dans le fichier Fonctions.java
